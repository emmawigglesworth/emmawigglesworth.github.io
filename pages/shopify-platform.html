Project spotlight

Shopify Merchant Insights

During my internship on Shopify's Merchant Services team I shaped internal dashboards that help teams act on real-time commerce metrics. I owned UX improvements end-to-end, from API design to component polish.

Front-end systems with intention

I delivered responsive React and TypeScript components that conformed to Polaris design tokens. Each addition was paired with Storybook documentation and accessibility checks so other squads could reuse the work confidently.

Data you can trust

  • Extended GraphQL queries to aggregate merchant health metrics in a single request.
  • Improved caching and pagination for heavy data sets, reducing perceived load time for internal users.
  • Expanded automated test suites using Jest and Cypress to protect critical journeys.

Outcomes

Stakeholders gained clearer visibility into merchant onboarding funnels, while support teams used the new dashboards to triage issues faster. The work also seeded patterns that informed broader design system updates.
