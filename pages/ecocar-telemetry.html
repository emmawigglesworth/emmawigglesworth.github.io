Project spotlight

EcoCAR Telemetry Suite

For McMaster Engineering's EcoCAR initiative I co-led the software stream developing connected vehicle analytics. The suite ingests telemetry data from hybrid vehicles and translates it into dashboards for engineering and leadership teams.

Pipeline architecture

Designed Python services that cleaned and transformed CAN bus data before persisting it in cloud storage. I introduced data validation steps to flag anomalies early and built monitoring alerts so we could respond quickly during track testing.

Insights for every stakeholder

  • Built dashboards that helped powertrain engineers analyse efficiency trends across drive cycles.
  • Provided leadership snapshots of charging habits and overall fleet health.
  • Shared documentation that onboarded new team members to the data model and tooling.

What I learned

The project strengthened my approach to data modelling, API design, and cross-disciplinary communication. Working closely with mechanical engineers pushed me to translate technical detail into actionable stories.
