Project spotlight

RBC Amplify Platform

As a Full Stack Developer on the RBC Amplify program, I helped extend an internal wealth management product that connects advisors and clients. Our squad delivered resilient new capabilities across Node.js microservices, React, and cloud automation.

Modernising core experiences

I collaborated with product and design partners to scope features that simplified how advisors review client portfolios. I implemented React components backed by GraphQL and REST endpoints, ensured accessibility compliance, and wired analytics events so teams could measure adoption from day one.

Operational excellence

  • Extended Node.js services with clear contracts and automated integration tests.
  • Introduced CI/CD checks that linted, tested, and deployed changes into cloud environments with one command.
  • Documented runbooks that helped incoming developers debug and monitor releases.

Impact

The enhancements improved advisor task completion times and reduced manual follow-up for the support team. By tightening feedback loops and codifying deployment playbooks, our squad set the groundwork for future cohorts to deliver confidently.
